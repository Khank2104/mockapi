# QuanTro - Motel Management System (ASP.NET Core MVC)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![.NET 8.0](https://img.shields.io/badge/.NET-8.0-blue.svg)](https://dotnet.microsoft.com/download/dotnet/8.0)

**QuanTro** is a professional property management solution designed for small to medium-sized motel owners. It provides a seamless end-to-end workflow from infrastructure setup and tenant onboarding to automated billing and real-time support requests.

---

## 🚀 Key Features

- **Dynamic Floor Map**: Visual room management supporting both multi-floor and single-floor properties.
- **Contract Management**: Professional 3-step wizard for contract creation and automated lifecycle tracking.
- **Smart Billing**: Automated meter reading (Electricity/Water) with prorated rent calculation and Excel export.
- **Tenant Portal**: Dedicated dashboard for tenants to view invoices and submit maintenance requests.
- **Background Tasks**: Automatic contract expiration handling and system cleanup.

## 🛠 Tech Stack

- **Backend**: ASP.NET Core MVC, Entity Framework Core (SQL Server / SQLite).
- **Frontend**: Bootstrap 5, Vanilla JavaScript (Modular ES6 style).
- **Architecture**: Service-Oriented Architecture (SOA), DTO pattern, SOLID principles.

# QuanTro - Hệ Thống Quản Lý Trọ (ASP.NET Core MVC)

**QuanTro** là giải pháp quản lý nhà trọ chuyên nghiệp dành cho các chủ trọ quy mô vừa và nhỏ. Hệ thống cung cấp luồng công việc xuyên suốt từ thiết lập hạ tầng, tiếp nhận khách thuê, đến tự động hóa thanh toán và xử lý yêu cầu hỗ trợ theo thời gian thực.

## 🚀 Các Tính Năng Chính

- **Sơ đồ Phòng & Tầng**: Quản lý trực quan theo dạng lưới, hỗ trợ nhiều kiểu khu trọ khác nhau.
- **Quản lý Hợp đồng**: Quy trình ký kết 3 bước chuyên nghiệp và tự động theo dõi vòng đời hợp đồng.
- **Hệ thống Hóa đơn**: Ghi chỉ số điện nước gối đầu, tự động tính tiền theo thời gian ở thực tế và xuất hóa đơn Excel.
- **Cổng thông tin Khách thuê**: Dashboard riêng cho người thuê xem hóa đơn và gửi yêu cầu sửa chữa.
- **Tác vụ Ngầm**: Tự động hóa việc xử lý hợp đồng hết hạn và dọn dẹp hệ thống.
